# Botanie_fyp
This is an plant recognition application with gamification. This apps is built with react native and firebase.

# Tools
- Visual Studio Code
- Expo

# Api Key Required
- Firebase
- Plant.id
- OpenWeatherMap

# How to run
- Node.js and Expo is installed

1. Download and extract the code.
2. Open terminal and direct to the code.
3. Run 'npm install'.
4. Run 'expo start' or 'npm start'.
